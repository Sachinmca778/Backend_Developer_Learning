## Mindset Reset

> "Negotiation = **business conversation**, not begging. Company invested 5-10 hours interviewing — they **want you to sign**."

### Key facts

- 80%+ of candidates accept first offer
- Companies typically have 10-25% negotiation room
- **Asking does NOT rescind offers** in 99% cases
- Recruiter ka job = close you (not lowball you)
- Non-negotiation = leaving 1-3 lakh+ on table

### Common fears (all overblown)

| Fear | Reality |
|------|---------|
| "Offer rescind ho jayega" | Almost never; only if rude/unprofessional |
| "Recruiter mujhse naraz hoga" | Recruiter expects negotiation |
| "Mein deserve nahi karta" | Market rate ≠ what you deserve; just data |
| "Company decision final hai" | Senior recruiters always have approval room |

## Market Research — India 2024-26

### Sources (use 2-3)

| Source | Strength |
|--------|----------|
| **levels.fyi** | Best for product / FAANG-level companies (USD-denominated mostly) |
| **Glassdoor** | Wide coverage, can be noisy |
| **AmbitionBox** | India-focused, decent for service / mid-sized |
| **LinkedIn Salary** | Aggregate ranges by role + location |
| **Naukri.com** | Job listings show ranges (not always accurate) |
| **Discord/Slack communities** | Real engineer conversations |
| **Levels India / Blind India** | Anonymous salary chatter |

### India backend ranges (2024-26 ballpark)

> ⚠️ Rough ranges. Verify per company tier + city + your stack.

| Experience | Service Co (TCS, Infy, mid-tier) | Product Mid-Tier (PayTM, Swiggy, etc.) | Product Top-Tier (Google, Microsoft, Atlassian, Adobe, etc.) |
|------------|---------------------------------|----------------------------------------|---------------------------------------------------------------|
| Fresher (0 yr) | 3-6 LPA | 8-15 LPA | 22-35 LPA |
| 1-2 yr | 4-8 LPA | 10-20 LPA | 25-45 LPA |
| **2-3 yr** | **6-12 LPA** | **14-28 LPA** | **30-55 LPA** |
| 3-5 yr | 10-18 LPA | 22-40 LPA | 45-80 LPA |
| 5-8 yr | 18-28 LPA | 35-60 LPA | 70-1.5 Cr |
| 8+ yr | 25-45 LPA | 50-90 LPA | 1-3 Cr |

→ **LPA = Lakh per annum = total CTC**.

### Tier categorization

- **Service** — TCS, Infosys, Wipro, Cognizant, Capgemini, HCL — slower growth, stable
- **Product mid-tier** — Indian unicorns (Razorpay, PhonePe, CRED, Zerodha, Swiggy, Zepto, Meesho, etc.)
- **Product top-tier** — FAANG India offices, Atlassian, Adobe, SAP Labs, Microsoft, Google, Amazon

→ Compensation can be **2-5x different** between tiers for same role.

### City multipliers

- Bangalore / Hyderabad / Mumbai / Delhi NCR → reference rate
- Pune / Chennai → ~5-10% lower
- Tier-2 cities → 15-25% lower
- Remote → varies (sometimes Tier-1 rate, sometimes lower)

## Knowing Your Worth — 1.8 yr PHP + Java Learning

### Honest baseline

```
Current: PHP backend, 1.8 yr exp
Target: Java backend role (transition)
```

### Realistic ranges

| Type of Co | Likely range |
|-----------|--------------|
| Service Co Java role | 5-9 LPA |
| Mid-tier product | 8-16 LPA (with strong projects + interview) |
| Top-tier product | 15-30 LPA (need strong DSA + system design) |

### Factors that **increase** offer

- ✅ Strong Java/Spring Boot portfolio (GitHub)
- ✅ Microservices project (Kafka / Spring Cloud)
- ✅ DSA strong (LeetCode 200+ medium solved)
- ✅ Existing offer in hand (leverage)
- ✅ Notice period short (immediate joiner premium)
- ✅ Quantified PHP achievements (transferable)
- ✅ Currently working at well-known co
- ✅ Tier-1 college (still matters in India)

### Factors that **decrease** offer

- ❌ "I don't have Java production experience" said upfront
- ❌ Long notice period (90 days)
- ❌ No GitHub / projects
- ❌ Career gaps unexplained
- ❌ Job-hopping pattern (many < 1 yr stints)

### Frame your transition

> "Backend engineer with 1.8 yr building production systems (PHP/Laravel). REST API design, microservices concepts, DB modeling, Redis caching, queue-based async — all transferable. Last 8 months focused on Java + Spring Boot ecosystem with structured projects."

→ **Sell the engineer, not the language.**

## When to Share Number — Defer Strategy

> "Pehla number share karne wala loses. **Defer kar sako utna defer karo.**"

### Why?

- They typed a budget already
- Your number = ceiling for negotiation
- Their offer = your floor

### Phases

```
1. Application form           — Often required field. Skip / put 0 / "negotiable" / range
2. Recruiter screen call      — Often asks "expectations"
3. Tech rounds                — Don't bring up
4. Offer stage                — Now negotiate
```

### Defer scripts (recruiter screen)

#### Script 1 — Soft defer

```
Recruiter: "What are your salary expectations?"

You: "I'm focused on finding the right role + team fit first. I'd love to learn more
about the position and the comp band you have in mind for this level. Once we both
see fit, we can align on numbers — what's the range you've budgeted for this role?"
```

#### Script 2 — Reverse the question

```
"Honestly, market data has a wide range for backend engineers in <city>.
What's the band the team has in mind so I can confirm we're in the same range?"
```

#### Script 3 — Range with research

```
"Based on my research and conversations with peers in similar roles, I'm seeing
ranges from <X> to <Y> LPA total comp. Where does this role sit?"
```

→ **Wide range is OK** (e.g., "12-25 LPA based on seniority"). Forces them to share.

### When forced to give a number

- Give **range** never single number
- Give **fixed CTC** as floor (e.g., "I'm looking for 14-20 LPA total comp")
- Anchor higher (top of range you'd accept)

## Total Compensation Components

> "Most candidates negotiate ONLY base salary. **Total comp = much more.**"

### India (private companies)

| Component | What |
|-----------|------|
| **Fixed (base)** | Monthly salary × 12; provident fund, gratuity included or excluded |
| **Variable** | Annual bonus, performance-linked (often 10-30% of fixed) |
| **ESOP / RSU** | Equity (startups: ESOP; listed: RSU) — usually vests over 4 yrs |
| **Sign-on bonus** | One-time joining bonus (1-3 lakh typical for mid-level) |
| **Retention bonus** | Year-2 / Year-3 lump sums |
| **Relocation** | If moving cities |
| **Health insurance** | Self + family coverage value |
| **Internet / WFH allowance** | Monthly |
| **Annual leave** | 20-30 days standard |

### Total Cost to Company (CTC) breakdown — typical

```
Total CTC = Base + Variable + Benefits + Equity (annualized)
```

Example breakdown for ~20 LPA mid-tier offer:

| Component | Amount |
|-----------|--------|
| Fixed (base) | 16 LPA |
| Variable bonus (15%) | 2.4 LPA |
| Sign-on (one-time) | 1.5 lakh |
| ESOP (4 yr vesting, 8 lakh total) | ~2 LPA |
| **Total Comp Year 1** | **~22 LPA effective** |

→ Negotiate any component — base, sign-on, equity, joining bonus.

## Negotiation Flow After Offer

### Step 1 — Receive offer

```
Recruiter: "We'd like to extend an offer. Total CTC 14 LPA — base 12 LPA, 
variable 2 LPA. Could you confirm by EOD tomorrow?"
```

### Step 2 — DON'T accept on call

```
You: "Thanks so much, really excited about the team. This is a big decision
for me — let me review the details and get back to you in 2-3 days. Is that OK?"
```

→ **Buy time**. Never decide in the moment.

→ "EOD tomorrow" = artificial pressure. 2-3 days standard reasonable.

### Step 3 — Get offer in writing

Email: "Could you share the formal offer letter / written breakdown so I can review?"

→ Don't negotiate verbally only. Numbers in writing.

### Step 4 — Research + decide your ask

- Compare to market data
- Decide: target number + walk-away number
- Check: are you talking to **other companies**? If yes, mention.

### Step 5 — Counter-offer

Email or call. (See scripts below.)

### Step 6 — They respond

Possible outcomes:
- ✅ Match your ask → accept
- 🟡 Partial bump → decide if good enough
- 🔴 "We can't" → decide if walk away

### Step 7 — Final acceptance

```
"Thanks for the revised offer. Happy to accept at 18 LPA fixed + 2 LPA sign-on.
Could you please send the updated offer letter? I'll sign + return today."
```

### Step 8 — Don't burn bridges if walk away

```
"Really appreciate the time and offer. Unfortunately at this comp level it doesn't
match my goals — I have to decline. Would love to stay in touch for future roles."
```

## Counter-Offer Scripts

### Script 1 — Standard counter (Hinglish + English)

```
Subject: Re: Offer — Backend Engineer position

Hi <Recruiter>,

Thanks again for the offer — really excited about <team / project>.

After reviewing the role + comp, I want to share some thoughts. Based on:
- My current package + market data for backend engineers with similar skill set
- Other conversations I'm having (one offer in late-stage)
- The scope of role you've described

I was hoping we could close the gap on base. Could the team consider 17 LPA fixed
(vs offered 14)? Variable + sign-on can stay similar.

I'd like to formally accept this week — really keen on joining the team. Let me know
how the team can work this out.

Thanks!
Sachin
```

### Script 2 — Phone (after first offer)

```
You: "Thanks for the offer — I appreciate the team's confidence in me.

I want to be transparent — based on my research and other conversations, the comp
at this level for similar roles is in the 17-20 LPA range. Is there room to revise?
I'd be happy to lock in this week if we can get to 18 LPA fixed."

[silence — let recruiter respond]

Recruiter: "Let me check with the team and get back to you."

You: "Thank you — looking forward to hearing back."
```

→ **Silence is your friend** post-ask. Don't ramble.

### Script 3 — Negotiating sign-on / equity instead

```
"I understand base is constrained. Could we explore:
- Higher sign-on bonus (e.g., 2.5 lakh instead of 1)?
- More equity grant?
- Faster review cycle (6 months instead of annual)?

These would also work for me to bridge to my target total comp."
```

### Script 4 — Walk away politely

```
"Thanks again — at this comp level, I can't make it work. If the team has
flexibility to revisit at <X LPA>, I'd love to discuss. Otherwise, I'd need to
respectfully decline. Either way, thank you for the time."
```

→ Sometimes recruiter comes back. Sometimes not. Both fine.

## Multi-Offer Leverage

> "Best leverage = **another offer in hand**."

### Strategy

1. **Time your interviews** — accept 1-2 weeks for offer expiry; use to interview elsewhere
2. **Be honest about other interviews** — "I'm in late-stage with another company"
3. **Specific numbers** if you have written offer:

```
"I have a competing offer from <Co X> at 18 LPA total comp. I'd prefer your team
because <reason>. Can we close the gap?"
```

### Don't

- Lie about offers — gets caught (small recruiter network)
- Reveal exact competitor name unless you must
- Play companies against each other aggressively → recruiters notice

### If only 1 offer

Less leverage, but still:
- Use market data
- Use "currently exploring others"
- Use scope mismatch ("I expect higher given this scope")

## India-Specific Tips

### Notice period

- 30 days = strong leverage
- 60-90 days = pressure on you (companies want fast joiners)
- **Buy-out** = company pays your current company in lieu of notice

→ Negotiate notice buy-out as part of offer (~1-3 lakh typical).

### Joining bonus

Most companies offer 50K-3 lakh sign-on for mid-level. **Specifically ask** if not offered.

### "We don't negotiate" line

Companies that say this:
- Sometimes true (Amazon India sometimes)
- Often soft "no" — push politely once

```
"I understand the framework. Even a 5-7% adjustment on base would make this a
clear yes for me — would the team have approval room there?"
```

### Gross CTC vs In-Hand

```
CTC          = What company spends on you (incl. variable, EPF, gratuity, insurance value)
Take-home    = Monthly bank credit after all deductions
```

Rough conversion (no other income):

| CTC | Take-home/month (approx) |
|-----|---------------------------|
| 10 LPA | ~62-68K |
| 15 LPA | ~88-95K |
| 20 LPA | ~1.10-1.20 lakh |
| 30 LPA | ~1.55-1.75 lakh |

→ Use online "CTC to in-hand" calculators (ClearTax, Cleartax, ICICI calc) for exact.

→ Negotiate **CTC** (mostly), but be aware of **take-home**.

### Variable / bonus

- Always ask: "What's the historical payout %?"
- 15% variable means full payout = 15%; usually 70-100% paid
- Don't count variable as guaranteed

### ESOP / RSU

- ESOP (private startup) — paper value till liquidity event (IPO / acquisition)
- RSU (public co) — actual stock vesting; sellable
- Vesting cliff: usually 1-yr cliff + monthly/quarterly vesting

### Tax bracket

Above 15 lakh → 30% slab — every additional lakh = 70K take-home → factor in.

## Red Flags

| Behavior | What it means |
|----------|---------------|
| "We don't negotiate" said firmly first call | Sometimes true, often "soft no" |
| Pressure to accept in 24h | They lack alternatives (good leverage) OR scammy |
| Verbal only, no written offer | Don't proceed |
| "We pay below market because [vague]" | Reconsider |
| Vague variable terms ("up to 30%") | Treat as zero |
| No salary breakdown given | Demand it |
| Recruiter dodges total-comp Q | They're hiding components |

## Common Output Traps

### Q1. Sharing current salary first

"Mera current 8 LPA hai" → recruiter offers 10 LPA (25% bump) — leaves money.

→ **Defer** + use market data instead.

### Q2. Single number expected

"I expect 12 LPA" — anchored low, no room.

→ Range + range high.

### Q3. Negotiating only base

→ Sign-on + equity + leaves + WFH all on table.

### Q4. Accepting on the call

→ Always ask for time + written offer.

### Q5. Burning bridges

→ Even on walk-away, polite. Recruiters have long memories + small networks.

### Q6. Counting unvested ESOP as cash

→ Discount heavily for non-listed; count public RSU close to face value.

---

## Pitfalls

1. **Sharing current CTC** unnecessarily — anchors low.
2. **Verbal-only offers** — get in writing.
3. **No counter** — leaving 1-3 lakh on table guaranteed.
4. **Lying about other offers** — caught easily.
5. **Aggressive tone** — "match or I leave" — usually backfires.
6. **Negotiating after signing** — too late.
7. **Ignoring variable risk** — counted as guaranteed but isn't.
8. **Forgetting notice buy-out** — could be 1-2 lakh extra.
9. **Decided in heat** of phone call.
10. **Skipping research** — going by gut feeling.

## Cheat Sheet

| Phase | What |
|-------|------|
| Pre-offer | Defer numbers; research market |
| Offer received | Don't accept on call; ask for written + 2-3 days |
| Counter | Polite, data-driven, range |
| Final | Confirm written offer; thank everyone |

| When asked salary expectation | Response |
|-------------------------------|----------|
| Application form | Skip / "negotiable" / range |
| Recruiter screen | "What's your range first?" |
| If forced | Wide range, anchor high |
| Post-offer | Now negotiate firmly |

| Negotiable | Items |
|------------|-------|
| Base salary | Most asked |
| Sign-on bonus | Often easier than base |
| Equity | Often flexible |
| Joining date | Notice buy-out negotiable |
| Title / level | Sometimes |

| Walk away if | Stay if |
|--------------|---------|
| Total comp < market floor | Within market band + good team |
| Vague terms | Clear written offer |
| Disrespectful negotiation | Constructive negotiation |

## Practice

1. Look up your role on **levels.fyi + AmbitionBox + LinkedIn Salary** — write down range.
2. Calculate your "walk-away minimum" + "stretch target" for next role.
3. Practice 3 deferral scripts out loud (record yourself).
4. Mock interview with friend playing recruiter — pressure salary discussion.
5. Get written offer (when you do) → review with senior friend before signing.
